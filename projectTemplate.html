ISWA Lebanon Website.

This project consisted of the the complete overhaul of the Integrated Solid Waste Management (ISWA) Lebanon's website. The main objective was to create a more easily updatable website with a clean and clear design. This would help ISWA Lebanon's members post content more regularly and in an easier manner, particularly new publications and events.
